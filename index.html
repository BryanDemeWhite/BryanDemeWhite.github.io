TTS Voice Wizard

Created as an accessibility feature for the Virtual Reality social platform VRChat. Although the app was particularly made for the deaf/hard of hearing and those that are mute, it boasts a ton of features for anyone to enjoy. This free and open-source project of mine started as a small hobby but quickly grew into an application used by hundreds of users daily. My app was even featured in the VR game's developer update. I am constantly adding new features as I receive feature requests from my users. - Subtitile your speech for the deaf. - TTS Voice for the mute. - Speech with "Speech to Text to Speech" - And much much more! ChoreBot

A discord bot created as apart of Programming Studio classes end of the semester project. The bots main purpose was to make doing chores fun. To achieve these we created a discord bot in which users can post chores like bounties and apon completion they would recieve points. Allowing users to claim the scoreboard of chores and spend points to get out of future chores. I was incharge of the entire frontend(making the bots responses to the users commands: options menu, embeded respones and messages) as well as much of the backend of the code. We used multiple API including the GIPHY API and LibreTranslate API.

Dungeon Runner RPG

A RPG game created with Construct3. Each room is randomly generated and can be a hallway to a encounter room of varying size. In addition created a combat system, shop, inventory system, item drop rarity system and a levelup/point system. There are even different tiers of weapons that drop as you level up. I created this project in my highschool game design class and it sparked the enjoyment for coding that I have today.

VRChat Avatar Calulator

VRChat is a game with many differing worlds to explore. I created this Calculator that can go on a users avatar and allow them to calculate basic math while in game. It really shines in escape room worlds where the user can do math without opening their desktop while in VR (which could slow down their escape time). The calculator is synced so that all users can see the values and even user the calculator off of another persons avatar. There is even a Desktop UI for the calulator so those without VR Headsets can still show eveyone their math calculations in real time.
